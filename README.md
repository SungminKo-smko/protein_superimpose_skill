# superimpose-protein 스킬

단백질 CIF 구조 파일의 superimposition(구조 정렬)을 위한 Claude Desktop 스킬입니다.
Claude Desktop에서 zip 파일로 설치하여 사용합니다.

## 개요

이 스킬은 단백질 구조 정렬 워크플로를 안내하는 가이드 역할을 합니다.
실제 정렬 연산은 별도의 MCP 서버가 수행합니다.

## MCP 서버 필요

이 스킬을 사용하려면 `protein_superimpose_mcp` MCP 서버가 설치 및 등록되어 있어야 합니다.
MCP 서버에 대한 자세한 내용은 아래 저장소를 참고하세요:

- https://github.com/SungminKo-smko/protein_superimpose_mcp
